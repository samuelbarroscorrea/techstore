# TechStore 🖥️

Projeto de site simples de uma loja de informática desenvolvido com HTML, CSS e Bootstrap.

## 📌 Sobre o projeto

O TechStore é um site fictício de uma loja de tecnologia, contendo páginas para:
- Página inicial com carousel
- Listagem de computadores
- Periféricos
- Promoções
- Formulário de cadastro de clientes

## 🧱 Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5

## 📄 Páginas do site

### 🏠 Início (index.html)
- Carousel com 5 imagens
- Destaques de produtos

### 💻 Computadores (pagina2.html)
- Grid responsivo com produtos
- Uso de col-lg-3

### 🎧 Periféricos (pagina3.html)
- Grid com 3 colunas
- Uso de col-lg-4

### 🔥 Promoções (pagina4.html)
- Grid responsivo:
  - Desktop: 4 colunas
  - Tablet: 2 colunas
  - Mobile: 1 coluna
- Preços com desconto e valor original riscado

### 📝 Cadastro (formulario.html)
Formulário completo com:
- Campos de texto (nome, CPF, e-mail, telefone)
- Data de nascimento
- Senha
- Select (estado, cidade, categoria)
- Radio button (sexo)
- Checkbox (interesses)
- Textarea (observações)
- Range (satisfação)
- Color (cor favorita)
- File (upload de imagem)

## 🎯 Objetivo

Projeto desenvolvido para prática de:
- Estrutura HTML
- Layout responsivo com Bootstrap
- Uso de Grid system
- Componentes de UI
- Formulários completos
